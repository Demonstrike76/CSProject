the Window module:

```

```	


contains:
	
tkinter window class:

```
<variableName> = window.window(<windowHeight>, <windowWidth>, <title>)
```

PygameWindow class:
	
Create PygameWindow:
```
<variableName> = window.PygameWindow(<windowWidth>, <windowHeight>, <title>)
```
